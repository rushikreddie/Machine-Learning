Overview

What's Inside

This repository is a hands-on study guide through the core pillars of classical machine learning. Each folder is a focused module — from fitting your first regression line to stacking ensembles and engineering powerful features. All notebooks are written to be readable, runnable, and reproducible.

Every module follows a consistent pattern: concept introduction → mathematical intuition → scikit-learn implementation → evaluation → real-world notes.

Repository Structure

Five Core Modules

📈
MODULE 01 · REGRESSION
Supervised Learning — Regression
Predict continuous outcomes. Covers linear and polynomial regression, gradient descent, cost functions, regularization (Ridge & Lasso), and performance metrics like RMSE and R².
LinearRegression Ridge Lasso gradient-descent
🔵
MODULE 02 · CLASSIFICATION
Supervised Learning — Classification
Classify data into discrete labels. Explores logistic regression, KNN, SVM, Naive Bayes, and decision trees with confusion matrices, ROC-AUC, precision, and recall.
LogisticRegression SVM KNN DecisionTree
🌲
MODULE 03 · ENSEMBLE
Ensemble Learning
Combine weak learners into powerful predictors. Covers bagging (Random Forest), boosting (AdaBoost, Gradient Boosting, XGBoost), and stacking strategies with voting classifiers.
RandomForest XGBoost GradientBoosting AdaBoost
📊
MODULE 04 · EVALUATION
Model Evaluation
Know if your model actually works. Implements cross-validation, k-fold splitting, bias-variance tradeoff analysis, learning curves, GridSearchCV, and hyperparameter tuning pipelines.
cross_val_score GridSearchCV k-fold bias-variance
⚙️
MODULE 05 · FEATURES
Feature Engineering
Turn raw data into model fuel. Explores encoding (one-hot, label, ordinal), imputation strategies, scaling (Standard, MinMax, Robust), feature selection, and pipelines for production-ready preprocessing.
OneHotEncoder StandardScaler Pipeline SelectKBest

Key Concepts

What you'll learn

Concept Description Module
Gradient Descent Iterative optimization to minimize the cost function by updating weights in the direction of steepest descent. 01
Overfitting & Regularization Techniques like L1/L2 penalties that penalize model complexity to improve generalization on unseen data. 01 · 04
Decision Boundary The hypersurface separating different class regions; visualized through 2D scatter plots in the notebooks. 02
Bagging vs Boosting Bagging trains models in parallel on random subsets; boosting trains sequentially, correcting prior errors. 03
ROC-AUC Area under the Receiver Operating Characteristic curve; measures classifier discrimination ability across thresholds. 02 · 04
Bias-Variance Tradeoff The tension between underfitting (high bias) and overfitting (high variance), visualized via learning curves. 04
Feature Scaling Normalizing or standardizing features so distance-based and gradient-based algorithms converge correctly. 05
Pipelines Chaining preprocessing and modeling steps into a single sklearn object for clean, leak-free workflows. 05
Getting Started

Run it locally in minutes

All notebooks run in a standard Python environment. Follow these steps to get up and running.

1
Clone the repository
Pull the repo to your local machine.
git clone https://github.com/rushikreddie/Machine-Learning.git
cd Machine-Learning
2
Create a virtual environment
Isolate your dependencies cleanly.
python -m venv venv
# macOS/Linux
source venv/bin/activate
# Windows
venv\Scripts\activate
3
Install dependencies
Everything you need in one command.
pip install numpy pandas matplotlib seaborn scikit-learn xgboost jupyter
4
Launch Jupyter
Open any notebook and start learning.
jupyter notebook
